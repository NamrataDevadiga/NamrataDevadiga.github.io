Mumbai Travel


In order to learn React I created a web app for a travel guide to the city of Mumbai. It included recommendations for all the best places to visit, food to eat, hidden gems for shopping and some of the best festivals to look out for in Mumbai.


I used images to enhance the beauty of the city and help users's to navigate to specific pages containing detailed information about the must visit spots.


I used white spaces and color contrast to highlight the important text. I also provided a navigation bar to help them easily access the different pages.


Apart from that I used the google maps api to help user's see the different spots on the map of Mumbai. This would help the user to create a better travel plan while keeping in mind the distance between the various places they would like to visit.
