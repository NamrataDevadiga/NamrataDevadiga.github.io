The Lazy Foodie


The Lazy Foodie is a website I developed to capture my culinary experience as an amateur chef. The main goal of this application was to provide simple but tasty recipes to other food enthusiasts. This application has been created using Angular, HTML, CSS and Bootstrap.


The app provides different types of recipes to users. Recipes are segregated based on categories such as Appetizers, Desserts, Meals and Snacks.


The use of white space and contrasting colors guides the user through the application. I have used a color scheme of red shades as it aligns well with the theme of the application i.e food.


For the recipes of individual dishes I have highlighted the important text using boldness and color contrast. Information is also grouped together to provide an ease of readability to the users. I have used images which enhance the look and feel of the application.


The application uses Bootstrap cards and Bootstrap grids to display content in a flexible manner.
