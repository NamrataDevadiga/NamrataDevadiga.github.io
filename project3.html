Mumbai Travel


Mumbai Travel is a website designed to act as a travel guide to the city of Mumbai. It includes recommendations for all the best places to visit, food to eat, hidden gems for shopping and some of the best festivals to look out for in Mumbai. This application was developed using React for the front-end and Python Flask Api and Sql database for the back-end.


Images were used to enhance the beauty of the city and help users navigate to specific pages containing more detailed information about the must visit spots or must do activities in the city.


White spaces and color contrast was used to highlight the important text on the pages. A top navigation bar was provided to help users easily access the different pages of the application.


The application also used the Google Maps Api to display to the users the different spots on the map of Mumbai. This feature was added to help the user to create a better travel plan while keeping in mind the distance between the various places they would like to visit in the city of Mumbai.
