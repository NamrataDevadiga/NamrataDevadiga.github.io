We Hire


We Hire was a website developed to help students and recruiters connect and interact at Career Fairs. The main goal of the application was to provide a platform to both parties to effortlessly communicate announcements and/or queries with one another. The application was created using HTML, CSS, Bootstrap, jQuery and Javascript for the User Interface and Python Flask Api and SQL Database for the back end.



The app provided the functionality to all users to view the list of recruiters and students registered for the event. It also gave them the option to view all messages sent and received by their own profile. jQuery and Socket.io was used to provide seamless communication between multiple users.



The website gave the users an opportunity to reply to messages in real-time. It showed the most recent messages at the top of the page to ensure that both students and recruiters were viewing the latest information at all times.
